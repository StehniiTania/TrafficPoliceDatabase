# *** Traffic police database ***
#********************************

Task: Create a traffic police database on fine receipts using a binary tree. The key will be the number of the car, the value of the node will be the list of offenses. If the receipt is added for the first time, then a new node appears in the tree, and data on the offense appears in the list; if not, the data is entered into the existing list. It is also necessary to implement
 the following operations:
• Output of the entire database (according to the numbers of cars and the list of offenses attributed to them);
• Data output by given number;
• Data output by number range;
• Saving and loading information from a file.



Works in VisualStudio 2019
